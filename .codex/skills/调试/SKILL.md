# start debug

已经打开的chrome浏览器使用的是远程调试，同时已经打开的浏览器拓展界面不会被加载出来，但是新打开的浏览器拓展主界面可以加载到

## Open New Dashboard Page

Open a new page to view the updated dashboard:

```
chrome-devtools_new_page
url: "chrome-extension://adglfigbkfnfemhgcdcmlkgbalfgcdlg/dashboard.html"
```

然后就可以使用chrome-devtools-mcp的其他功能进行调试了。
